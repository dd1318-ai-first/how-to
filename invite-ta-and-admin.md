# Adding a TA or Admin to the Classroom

To add someone as admin or teaching assistant to AI First Classroom:

## Add user as to GitHub Org as an owner

1. Goto https://github.com/orgs/dd1318-ai-first/people
2. Click **Invite as owner**
3. Add GitHub username
4. Select **Owner** as role

## Share GitHub Classroom Url with user

1. Send URL https://classroom.github.com/classrooms/254276161-dd1318-ai-first-2026 to complete connection.
