# Create and Share a Lab Assignment

To create a new lab assignment in the AI First 2026 GitHub Classroom, follow these steps:

## Create a New Repository in the Organization

1. Go to the GitHub organization: https://github.com/dd1318-ai-first/
2. Click on the **New** button to create a new repository
3. Name the repository according to the assignment (e.g., `lab-1` or `ovn-1`)
4. Set the repository to **Private**
  - this will stop students from seeing the repo and potentially changing it
5. In **Start with a template**, select `dd1318-ai-first/mll-agent`
  - this will setup the MLL agent in the correct way and is highly recommended
6. Click **Create reposistory**
7. In the repo **Settings**, check the **Template repository** option (immediately below the **Repository name** field)
  - this is needed to create a Classroom assignment
8. **Edit** the README file to include your instructions for the assignment
9. OBS! The MLL Agent is added by default by using the template in step 5 (you can confirm this by observing a `.github/agents` folder in the repo files)

## Create the Assignment in GitHub Classroom

The final step is to create the assignment in GitHub Classroom:

1. Go to the GitHub Classroom: https://classroom.github.com/classrooms/254276161-dd1318-ai-first-2026
2. Click on **New Assignment**
3. In Assignment Basics, fill in the title (set deadline if needed) and select **Individual Assignment**
4. In Starting Code and environment, find the template repository you just created in the organization
  - OBS! the list seems cluttered to me so make sure to search for the fullpath `dd1318-ai-first/your-assignment`
6. Leave the visibility as **Private** so only students can access it
7. Ignore "Add a supported editor" option
8. Ignore the Grading and feedback section and click **Create Assignment**

## Testing the Assignment

After creating the assignment, you will be redirected to the assignment overview page. If you want to try the lab yourself, copy the invitation link on the assignment page into a new browser tab.

## Share the Assignment with Students

Copy the invitation link provided on the assignment page and share it with your students so they can join the assignment.

**OBS!** If students cannot access the assignment via the invitation link, ask them to check their email they made their GitHub account with as there may be an invivation there.
