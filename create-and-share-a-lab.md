# Create and Share a Lab Assignment

To create a new lab assignment in the AI First 2026 GitHub Classroom, follow these steps:

## Create a New Repository in the Organization

1. Go to the GitHub organization: https://github.com/dd1318-ai-first/
2. Click on the "New" button to create a new repository
3. Name the repository according to the assignment (e.g., `lab-01-introduction`)
4. Set the repository to private if you want only students in the classroom to access it
5. Initialize the repository with a README file
6. In the repo settings, check the **Template repository** option (immediately below the **Repository name** field)
7. Edit the README file to include instructions for the assignment

## Adding MLL Agent

To ensure that the MLL Agent is included in the assignment repository, follow these steps:

1. Create a new file in the repository named `.github/agents/mll-agent.md` (pay attention to the '.' at the beginning of the path)
2. Edit the `mll-agent.md` file and copy the contents of https://github.com/dd1318-ai-first/mll-agent/blob/main/.github/agents/mll-agent.md
3. Commit the changes to the repository

## Create the Assignment in GitHub Classroom

The final step is to create the assignment in GitHub Classroom:

1. Go to the GitHub Classroom: https://classroom.github.com/classrooms/254276161-dd1318-ai-first-2026
2. Click on "New Assignment"
3. In Assignment Basics, fill in the title (set deadline if needed) and select "Individual Assignment"
4. In Starting Code and environment, find the template repository you just created in the organization
5. Leave the visibility as private so only students can access it
6. Ignore "Add a supported editor" option
7. Ignore the Grading and feedback section and click "Create Assignment"

## Testing the Assignment

After creating the assignment, you will be redirected to the assignment page. If you want to try the lab yourself, copy the invitation link on the assignment page into a new browser tab.

## Share the Assignment with Students

Copy the invitation link provided on the assignment page and share it with your students so they can join the assignment.
